Projeto desenvolvido em React, feito para praticar conceitos fundamentais do front-end em um simulador de plataforma de avaliações de filmes e séries.

Ao ser acessado, o site apresenta uma página inicial interativa, com foco na experiência do usuário. A navegação fornece diferentes rotas, como:

- HOME - exibe a página inicial, onde é possível também ter acesso à sessão "TRENDING NOW" que indica os filmes mais populares no momento.
- CATALOG — mostra todos os filmes disponíveis no catálogo.
- REVIEW — rota para a qual o usuário é redirecionado ao clicar na capa de um filme ou pesquisá-lo pela barra de navegação.

Os dados utilizados em todas essas sessões são obtidos a partir de um arquivo JSON, que funciona como um banco de dados alternativo para armazenar informações dos filmes, como capa, título, descrição, ano e nota. Esses dados são consumidos dinamicamente pelos componentes do React.

O projeto é totalmente focado no front-end, servindo como um exercício de responsividade, componentização e interatividade. Foram utilizadas bibliotecas como React Slick para a criação de carrosséis dinâmicos e seções deslizáveis. O layout foi planejado para se adaptar bem a diferentes tamanhos de tela.

Conceitos aplicados: Componentização, Props e State, Hooks do React, consumo de dados via JSON, criação de rotas, design responsivo em CSS, uso de bibliotecas externas (React Slick), navegação dinâmica e boas práticas de organização de componentes.
